# Archive

This directory contains historical artifacts, deprecated code, and backups.

## Structure

- `YYYY-MM-DD-*/` - Dated snapshots (e.g., migration backups)
- `deprecated-code/` - Old implementations replaced by newer versions
- `historical-configs/` - Previous configuration versions
- `session-notes/` - Planning documents and session transcripts

## Retention Policy

| Type | Retention | Review Frequency |
|------|-----------|------------------|
| Migration backups | 30 days post-stability | After each migration |
| Deprecated code | Indefinite | Quarterly |
| Historical configs | Indefinite | Quarterly |
| Session notes | Indefinite | As needed |
